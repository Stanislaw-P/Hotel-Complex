# Hotel Complex Management System

Система управления гостиничным комплексом, разработанная в рамках учебного проекта. Приложение позволяет автоматизировать основные бизнес-процессы гостиницы: управление номерами, бронирование, регистрация гостей, учет услуг и формирование счетов.

## 📋 Функциональность

- **Управление номерами** - добавление, редактирование, просмотр статуса номеров
- **Бронирование** - создание и управление бронями для гостей
- **Регистрация гостей** - ведение базы постояльцев
- **Корпоративные клиенты** - работа с партнерами и договорами
- **Дополнительные услуги** - заказ и учет услуг (питание, прачечная, трансфер и др.)
- **Финансовый учет** - формирование счетов, учет оплат
- **Отзывы и жалобы** - обратная связь от гостей

## 🛠 Технологический стек

### Backend
- **ASP.NET Core MVC** (.NET 8) - основной фреймворк
- **MySQL** - реляционная база данных
- **ADO.NET** - прямое выполнение SQL запросов

### Frontend
- **Bootstrap 5** - адаптивный дизайн
- **HTML/CSS/JavaScript** - стандартные веб-технологии

## 🗄 Структура базы данных

Проект включает 12 взаимосвязанных таблиц:
- RoomType, Room - номера и их типы
- Guest, Employee - гости и сотрудники
- CorporatePartner, Contract - корпоративные клиенты
- Service, ServiceOrder - дополнительные услуги
- Booking, Stay - бронирования и проживание
- Invoice - счета
- Review - отзывы

## 🚀 Особенности реализации

- **Чистый SQL** - все запросы пишутся вручную, без использования Entity Framework
- **Автоматическое создание БД** - при первом запуске структура базы данных создается автоматически
- **Инициализация тестовыми данными** - база заполняется начальными данными для демонстрации
- **Репозитории** - изоляция логики доступа к данным
